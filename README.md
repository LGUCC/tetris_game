> How to run

[sfml setup guide](https://www.sfml-dev.org/tutorials/2.5/start-vc.php)

## Windows:- 
```
 1. Open Terminal here
 2. Run _ g++ cli.cpp -o setup
 3. ./setup.exe
```

## Linux:-
```
 1. Open Terminal here
 2. g++ -c main.cpp
 3. g++ main.o -o sfml-app -lsfml-graphics -lsfml-window -lsfml-system
 4. ./sfml-app 
```
